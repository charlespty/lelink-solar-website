## 🚀 技术栈

### 前端技术
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **图标**: Lucide React, Heroicons
- **UI组件**: Headless UI, 自定义组件

### 后端技术
- **运行时**: Node.js
- **框架**: Express.js
- **数据库**: MongoDB + Mongoose
- **认证**: JWT
- **邮件**: Nodemailer + SendGrid
- **文件上传**: Cloudinary
- **验证**: Joi

### 部署和运维
- **前端部署**: Vercel (推荐)
- **后端部署**: Railway/Heroku
- **数据库**: MongoDB Atlas
- **CDN**: Cloudflare
- **监控**: Google Analytics 4
- **错误跟踪**: Sentry (可配置)

## 🚀 部署选项

### 前端部署 (Vercel - 推荐)
```bash
# 1. 安装 Vercel CLI
npm i -g vercel

# 2. 登录 Vercel
vercel login

# 3. 部署到生产环境
cd website
vercel --prod
```

### 后端部署 (Railway)

2. 配置环境变量
3. 自动部署
